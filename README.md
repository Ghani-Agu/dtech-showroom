# Dtech Showroom

Cinematic 3D showroom for Dtech Algérie's product catalog.

- **Path:** D — The Catalog as Cinematic Showroom
- **Phase:** 1.0 — Foundation scaffold
- **Stack:** Next.js 15 · TypeScript strict · Tailwind v4 · React Three Fiber · Framer Motion · GSAP · Lenis · Drizzle · Postgres

## Development

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000
